# Jump up

Unity 공부용 프로젝트

<br>

## Rule

1. 흰색 공은 계속 바닥으로 떨어지며, 위와 양 옆 벽과 충돌한다.
2. 플레이어는 입력을 통해 선을 그을 수 있고, 선은 흰색 공과 충돌한다.
3. 공이 화면 바닥에 닿으면 gameover
4. 공이 살아있는 채로 1분을 버티면 Clear
5. 주기적으로 사라지는 까만 점이 스폰되며 흰색 공이 까만 점이 사라지기 전에 점을 먹으면 1 Point
